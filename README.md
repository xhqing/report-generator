# report-generator

MCP Server for generating market research reports in HTML format.

## Features

- Generate HTML research reports with parameterized templates
- Support empty template generation (placeholder structure)
- Compliant with financial research report formatting standards
- Includes: market data tables, event analysis, index/stock/ETF analysis, reasoning chains, references, disclaimers

## MCP Tools

| Tool | Description |
|------|-------------|
| `generate_report` | Generate a complete HTML research report from input data |
| `generate_empty_template` | Generate an empty report template with placeholders |
| `get_report_schema` | Get the JSON schema for report input data |
